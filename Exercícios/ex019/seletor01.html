Identificando id

Para identificar um ID precisa colocar a # na frente.

Exemplo=h1#principal

(obs) Não necessariamente precisa colocar a tag que no caso foi o h1 pode só colocar #principal

Outra coisa,se em HTML é ID,em CSS é #

Se em HTML é class,em css é .

Diferença entre id e class (e outros negocinhos)

Basicamente,o id é único e o class pode ser usado mais de uma vez

A tag span serve pra você marcar uma frase ou uma palavra específica e criar uma class para mudar especificamente ela.

Nada impede vocÊ colocar duas classes na mesma frase ou palavra para isso basta dar um espaço entre uma classe e a outra
